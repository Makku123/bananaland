Enhance the app by creating a user registration and login system.

For now, use SQLite as the backend database that will store user profiles
and other metrics that will be tracked per user (e.g. games played, wins,
losses, karma, etc.).

The initial regstration process will be simple and will only require the following fields:

- Nickname
- Email
- Password

Later we will enhance the registration system and add support for 3rd party authentication
providers (e.g. Google, Facebook, etc.) and also add support for additional fields
(e.g. profile picture, bio, etc.).
